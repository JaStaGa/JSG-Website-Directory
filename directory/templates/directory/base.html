2K MyPlayer Builder

With the price that NBA2K VC costs nowadays, you need your first build to be 'the one'. This inspired me to create this Django web app. It allows users to select desired NBA2K25 badges and compute the required attributes and height to unlock them. Built with Django models, forms, and class-based views; data stored in SQLite/Postgres; styled with Bootstrap and custom CSS.

FriendZone

A mock social media app supporting user registration, profiles, status posts, image uploads, and friend connections. Built with Django’s auth system, FileField for media, and ListView/DetailView.

Launch App

Voter Analytics

A dynamic data-driven application analyzing US election results. Integrates Django ORM with Pandas for data processing, and renders interactive charts via Plotly. Supports filtering by state and year.

Launch App

Random Quote Generator

A simple API-powered app that displays random inspirational quotes. Backend in Django REST Framework, frontend using vanilla JS and AJAX. Quotes stored in a PostgreSQL database and served via JSON endpoints.

Launch App

Fictional Restaurant App

A CRUD application for managing a fictional restaurant’s menu. Demonstrates Django forms, models, and auth: users can add/edit menu items, leave reviews, and upload images. Styled with Tailwind CSS.
