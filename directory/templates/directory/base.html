2KHub

MyPlayer Build Planner

Tired of the VC grind? 2KHub helps you plan the ultimate NBA2K25 MyPlayer build before you even spend a dime. Select your dream badges and it calculates the exact height and attribute requirements. Built with Django models, forms, and class-based views; data smoothly transitions between SQLite locally and PostgreSQL in production; styled with Bootstrap and custom CSS for a polished, responsive experience.

Launch App

FriendZone

Social Sandbox

Ever wanted a lightweight social network just for fun? FriendZone lets you sign up, craft your profile, post status updates with images, add friends, and explore a personalized news feed. Powered by Django’s authentication and FileField for media storage, it leverages ListView and DetailView for clean, maintainable code.

Launch App

Voter Analytics

Election Insights

Dive into US election data with Voter Analytics. I combined Django ORM with Pandas for robust data processing and rendered interactive Plotly charts to visualize trends by state and year. Filter and explore key metrics in real time—perfect for data enthusiasts.

Launch App

Lil Wayne Quote Machine

Random Quote Generator

Get inspired by random Lil Wayne lyrics on demand. This app uses Django REST Framework to serve quotes stored in PostgreSQL via JSON endpoints, and a vanilla JavaScript front end makes AJAX calls to pull fresh quotes without page reloads.

Launch App

Sanji’s Kitchen

One-Piece Inspired Restaurant

Step into the Grand Line with Sanji’s Kitchen, a themed restaurant app that lets you browse a custom menu, place orders, and enjoy chef-crafted dishes. Showcases Django forms, models, and authentication, wrapped in Tailwind CSS for a sleek, modern interface.
